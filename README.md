## Datasets

The self-constructed datasets used in this paper (**KFWalls** and **Pota**) are too large to be hosted directly on GitHub. You can download the complete `dataset.zip` from the following link:

* **Baidu Netdisk (百度网盘):** [Download dataset.zip](https://pan.baidu.com/s/1bpI1W8D74M_A8VVnUu0nTA?pwd=2fes)
* **Extraction Code (提取码):** `2fes`

### Folder Structure

After downloading and extracting `dataset.zip`, please ensure the folder structure is organized as follows before running the code:

```text
dataset/
├── KFWalls/
│   ├── train/
│   │   ├── images/
│   │   └── masks/
│   └── test/
│       ├── images/
│       └── masks/
└── Pota/
    ├── train/
    │   ├── images/
    │   └── masks/
    └── test/
        ├── images/
        └── masks/
